# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Forjate, please report it responsibly.

**Do not open a public issue.**

Instead, email **security@aitizate.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We'll acknowledge your report within 48 hours and work with you on a fix before any public disclosure.

## Scope

This policy covers the Forjate repository and its components. It does not cover third-party applications deployed through Forjate (Traefik, PostgreSQL, etc.) — report those upstream.

|---------|-----------|
